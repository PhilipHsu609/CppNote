# Modern C++ Notes

A study reference for Modern C++ (C++11 to C++23).

## View Online

Visit the [documentation site](https://philiphsu609.github.io/CppNote/) (deployed via GitHub Pages).

## Local Development

```bash
quarto preview   # Live preview at localhost:4000
quarto render    # Build to _site/
```

## Topics

| Section | Content |
|---------|---------|
| Core | Types, const, constexpr, enums, structs, functions, lambdas |
| Classes | Constructors, inheritance, virtual, operators, RAII |
| Templates | Deduction, specialization, SFINAE, variadic, metaprogramming |
| Memory | Allocation, memory ordering |
| Stdlib | Containers, optional, variant, chrono, filesystem, I/O |
| Concurrency | Threads, mutex, atomics, condition variables, async |
| Compilation | Linkage, inline, translation units |
| Patterns | SOLID, factory, builder, singleton, prototype, adapter, pimpl |
| Third-Party | fmt, Google Test |
